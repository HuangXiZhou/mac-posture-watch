# Support

Use GitHub issues for reproducible bugs and focused feature requests.

Do not share real API keys, Bark endpoints, camera captures, debug frames, logs with
secrets, or baseline files. Redact command output before posting it.

For setup problems, include:

- macOS version;
- Python version;
- install command used;
- redacted output from `posture-watch doctor --camera`;
- whether the issue happens with `ENABLE_LLM_VERIFY=0`.
